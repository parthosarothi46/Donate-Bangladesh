Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a webpage as a tree of objects. It allows developers to interact with and modify the content, structure, and style of a webpage using languages like JavaScript. The DOM also handles events, making web pages dynamic and responsive to user actions.

How do you select an element from the DOM?

getElementById('id') selects by ID. getElementsByClassName('class') selects by class (live collection). getElementsByTagName('tag') selects by tag (live collection). querySelector('selector') selects the first match. querySelectorAll('selector') selects all matches (static collection).

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events for its child elements. It's useful because it improves performance by reducing the number of listeners and works for dynamically added elements. It leverages event bubbling to manage multiple elements with a single listener.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes in the DOM, use setAttribute() or directly modify properties like element.src. For styles, use the style property or modify classes with classList.add() and classList.remove().
